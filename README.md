# ProyectoFinal Computación Gráfica  
## 2026-1 Equipo15

### ¿Cómo descargar el proyecto?

Para descargar el proyecto en su equipo local siga las siguientes instrucciones:

1. Vaya al enlace del repositorio en *GitHub*: https://github.com/yo-fer/ProyectoFinal_ComputacionGrafica_2026-1_Equipo15
2. Dé click en la pestaña "`<> Code`", seleccione la opción `HTTPS` y copie el enlace generado:
https://github.com/yo-fer/ProyectoFinal_ComputacionGrafica_2026-1_Equipo15.git
3. Ingrese a una terminal. Asegúrese de tener `Git` instalado.
4. Posiciónese dentro del directorio en el que desea clonar el repositorio e ingrese el siguiente comando:

`git clone <enlace_copiado_en_paso_2>`

La descarga de todas las carpetas del repositorio comenzará.

5. Una vez clonado el repositorio, puede abrir la carpeta principal (`ProyectoFinal_ComputacionGrafica_2026-1_Equipo15`) a través del comando `cd` o usando el explorador de archivos. 
6. Finalmente, para abrir el proyecto en `Visual Studio`, seleccione la opción **Abrir un proyecto o una solución** y seleccione el archivo con extensión `.sln`.
7. El proyecto abrirá dentro de `Visual Studio` y ahora será capaz de compilarlo. 

Opcionalmente puede seleccionar la opción **Clonar un repositorio** y seguir los pasos descritor por el IDE, sin embargo, recomendamos que siga la primera opción 
para evitar problemas en la descarga de la carpeta *External Libraries*.

### ¿Cómo ejecutar el proyecto?

Una vez que tenga el proyecto abierto en *Visual Studio*, en la pestaña de **Explorador de soluciones** seleccione el archivo principal con extensión `.cpp`,
posteriormente, en la parte superior observerá un par de opciones. La primera de ellas es un menú de opciones que le permite seleccionar `Debug` o `Release`, 
deje **Debug** como opción. A un lado verá otro menú que le permite seleccionar `x64` o `x86`, en este caso, seleccione **x86**.

Finalmente, presione el botón `Depurador local de Windows`. El proyecto comenzará a compilar y al terminar se ejecutará.
